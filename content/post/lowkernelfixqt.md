``` shell
error while loading shared libraries: libQt5Core.so.5: cannot open shared object file: No such file or directory
```

fixed by

```shell
sudo strip --remove-section=.note.ABI-tag /usr/lib/x86_64-linux-gnu/libQt5Core.so.5
```
